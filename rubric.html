Grading

Grades with be determined according to the following:

  • Class Participation - 10%
  • Project Brief - 10%
  • Sitemap - 10%
  • Wireframes - 10%
  • Website Design in Figma (Mid term) - 25%
  • Final Website Code (Final) - 35%
